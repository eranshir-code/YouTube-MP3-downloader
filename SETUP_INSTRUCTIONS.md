# Dana Shir MP3 Downloader - Setup Instructions

## One-time Setup (only needed once)

### Step 1: Install Homebrew
1. Open **Safari** and go to: **brew.sh**
2. Copy the installation command from the website
3. Open **Terminal** (find it in Applications > Utilities)
4. Paste the command and press Enter
5. Follow the prompts (you may need to enter your password)

### Step 2: Get the app files
1. Download the folder from GitHub or copy it from USB drive
2. Put the folder anywhere you like (Desktop or Documents works well)

## How to Use

### Every time you want to download music:

1. **Double-click** the file called: `Dana_Shir_Downloader.command`
2. A window will open and your browser will automatically open to the app
3. Paste a YouTube URL and click Download
4. Your MP3 files will be saved to your **Downloads** folder

### To stop the app:
- Just close the Terminal window that opened

## Troubleshooting

**Q: The .command file won't open**
- Right-click the file → Open → Click "Open" in the security warning

**Q: It says dependencies are missing**
- Make sure you completed Step 1 (Install Homebrew)
- Double-click the file again - it will auto-install what's needed

**Q: Where are my downloaded files?**
- In your Downloads folder (Finder → Go → Downloads)

## Notes
- Downloaded files are named with the YouTube video title
- If lyrics are available, they're saved as .srt files
- You need internet connection to download
